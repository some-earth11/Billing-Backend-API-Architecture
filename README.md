# Billing Backend API Architecture
Node.js backend for E-Commerce Billing with Taxation with user/admin access.

- Create User
- Get all Products
- Create Product
- Get all Services
- Create Service
- Get all Cart Items
- Add to Cart
- Clear Cart
- Remove Item
- View all Orders (Admin)
